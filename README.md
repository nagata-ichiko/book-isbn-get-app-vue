# book-isbn-get-app
これは本のタイトルを元にISBNを取得するアプリです

読んだ本の管理がむずかしくなったため、読書系Webサービスに読んだ本を登録する際、

一括登録機能を使用するためにはISBNコードが必要だったため、作成しました。
